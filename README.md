# folio.js

## Features

- Supports a variety of document formats, including PDF, DOC, PPT, XLS and TXT.


## Get Started

### Install

```sh
npm install folio.js

# pnpm
pnpm add folio.js

```

### Usage

API:

```js
import { } from 'folio.js'
import * as folio from 'folio.js'


```

CLI:

```sh
npx folio

```
